# Smart Workplace AI

A privacy-first, AI-powered smart workplace platform designed to support productivity, HR workflows, and women-centered health and caregiving needs using **synthetic data only**.

## Key Features
- Work hours & project tracking
- AI-driven workload and scheduling recommendations
- Opt-in health and caregiving support
- Privacy-preserving HR integration
- Rule-based AI with future ML expansion

## Tech Stack
- Python (Flask / FastAPI)
- SQL (PostgreSQL/MySQL)
- Pandas, Scikit-learn (AI logic)
- JWT Authentication

## Data Ethics
All employee and health-related data used in this project is fictitious and generated for demonstration purposes only.
